운영 데이터 관리

가계정, 독서모임, 내서재, 아카이브, 북라운지를 JSON 한 파일로 관리합니다. 공모전 데모용 관리자 화면입니다.

BOOKMATE로 돌아가기

운영 데이터 JSON

수정 후 “브라우저에 적용”을 누르면 BOOKMATE 화면에 반영됩니다.

수정하는 곳

users : 가계정, 닉네임, 비밀번호, 프로필, 소속도서관

gatherings : 독서모임 목록, 참여 여부, 정원, 일정

recentBooks : 내서재 완독/기록 카드

recentArchives : 내 아카이브에 보이는 기록

loungeBookmates : 북라운지에 보이는 북메이트

주의

이 관리자 페이지는 데모용입니다. GitHub 파일 자체를 서버에서 직접 수정하지 않고, 현재 브라우저 저장소에 운영 데이터를 적용합니다. 적용 후 JSON을 다운로드해서 보관하면 다음 버전에도 다시 업로드할 수 있습니다.

빠른 사용법

  1. 기본 템플릿을 누릅니다.
  2. 닉네임, 모임, 아카이브를 수정합니다.
  3. 브라우저에 적용을 누릅니다.
  4. BOOKMATE로 돌아가 확인합니다.
